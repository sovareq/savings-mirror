caveman-besparing

periode oproepen opus-baseline werkelijk tier-winst caveman-winst totaal-winst % tokens-bespaard

tier-winst = real-gemeten (opus-prijs × tokens − werkelijke prijs × tokens). caveman-winst = schatting via mode-benchmark + lengte-heuristiek (clamp 20–85%). totaal = som beide. echte caveman-meting vereist controle-run zonder caveman.

≈ Anthropic list-prijs equivalent. Abonnementen betalen een vast maandelijks bedrag — deze bedragen tonen wat pay-per-token API-gebruikers met dezelfde activiteit zouden uitgeven.
